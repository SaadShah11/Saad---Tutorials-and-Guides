‚ö° Ultimate Zsh Setup Guide

For iTerm2 (macOS) and Ubuntu Terminal

1. Install Zsh & Oh My Zsh

Mac

Ensure you have Homebrew installed. Zsh is usually default on modern Macs, but it's good to update.

brew install zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Ubuntu

Install Zsh and set it as default.

sudo apt update
sudo apt install zsh -y
chsh -s $(which zsh)
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
‚ö†Ô∏è Note: You may need to log out and log back in for the default shell change to take effect.

2. Install Theme (Powerlevel10k)

This theme provides the "Git Branch" visuals and icons. It requires a specific font to look correct.

Both Install the Theme

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Both Install Fonts (Crucial!)

Download and install MesloLGS NF fonts. The theme will look broken without them.

  1. Download these four files:
    • MesloLGS NF Regular.ttf
    • MesloLGS NF Bold.ttf
    • MesloLGS NF Italic.ttf
    • MesloLGS NF Bold Italic.ttf
  2. Mac Double-click each file and click "Install Font".
  3. Ubuntu Move files to ~/.local/share/fonts and run fc-cache -fv.

Mac Configure iTerm2

Go to iTerm2 > Preferences > Profiles > Text and change Font to MesloLGS NF.

Ubuntu Configure Terminal

Open Terminal Preferences, go to your profile, check "Custom Font", and select MesloLGS NF.

Both Configure VS Code / Cursor

The integrated terminal in VS Code and Cursor uses a different font setting. To fix broken icons:

  1. Open VS Code or Cursor
  2. Press ‚åò + Shift + P (Mac) or Ctrl + Shift + P (Windows/Linux)
  3. Type "Open User Settings (JSON)" and select it
  4. Add this line inside the JSON object: 
    "terminal.integrated.fontFamily": "MesloLGS NF"
  5. Save the file and restart VS Code/Cursor
üí° Quick Alternative: Press ‚åò + , to open Settings, search for "terminal font family", and enter MesloLGS NF.

3. Install Productivity Plugins

Autosuggestions: Suggests commands as you type based on history (press right arrow to accept).
Syntax Highlighting: Colors commands green (valid) or red (invalid).

Both Run these commands:

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

4. The Configuration File (.zshrc)

Replace the content of your ~/.zshrc file with the configuration below. You can edit it by running nano ~/.zshrc.

# Enable Powerlevel10k Instant Prompt
if [[ -r "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh" ]]; then
  source "${XDG_CACHE_HOME:-$HOME/.cache}/p10k-instant-prompt-${(%):-%n}.zsh"
fi

# Path to your oh-my-zsh installation.
export ZSH="$HOME/.oh-my-zsh"

# Set name of the theme to load.
# Powerlevel10k provides the git status, icons, and speed.
ZSH_THEME="powerlevel10k/powerlevel10k"

# Standard plugins can be found in $ZSH/plugins/
# Custom plugins may be added to $ZSH_CUSTOM/plugins/
plugins=(
    git
    zsh-autosuggestions
    zsh-syntax-highlighting
    web-search       # Allows typing "google something"
    copypath         # Copies current folder path
    copyfile         # Copies file content
)

source $ZSH/oh-my-zsh.sh

# User configuration
export LANG=en_US.UTF-8

# Preferred editor for local and remote sessions
if [[ -n $SSH_CONNECTION ]]; then
  export EDITOR='vim'
else
  export EDITOR='nano'
fi

# Aliases - Productivity shortcuts
alias zshconfig="nano ~/.zshrc"
alias ohmyzsh="nano ~/.oh-my-zsh"
alias c="clear"
alias ..="cd .."
alias ...="cd ../.."

# To customize prompt, run `p10k configure` or edit ~/.p10k.zsh.
[[ ! -f ~/.p10k.zsh ]] || source ~/.p10k.zsh

5. Finalize Setup

  1. Restart your terminal (close and open iTerm2/Terminal).
  2. You should see the Powerlevel10k configuration wizard start automatically.
  3. If it doesn't start, type: p10k configure
  4. Follow the on-screen prompts to choose your preferred style (Rainbow, Lean, etc.) and time format.
  5. Enjoy! Type a command you used before to see the grey autosuggestion, then press ‚Üí (Right Arrow) to complete it.
üí° Tip: You can re-run p10k configure anytime to change your prompt style!
